
## Usage

1. Clone the repository to your local machine.
    ```sh
    git clone https://github.com/Gearz9/StudyNotion.git
    ```

2. Install the required packages.
    ```sh
    npm install
    ```

3. Start the development server.
    ```sh
    npm run dev
    ```
